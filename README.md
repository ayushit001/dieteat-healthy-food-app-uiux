Project Overview
This project presents a complete end-to-end UI/UX design for a health-focused food delivery platform, covering the full customer journey from onboarding to order placement.

## Key Features
- Personalized onboarding and dietary preference selection
- Home, menu browsing, and food discovery flows
- Cart, checkout, and order tracking experience
- Profile, preferences, and account management
- Consistent design system with a user-friendly interface

## Design Process
- User journey mapping
- Wireframing and interaction design
- UI consistency and usability optimization
- End-to-end flow prototyping using Figma

## Tools Used
- Figma (UI/UX design and prototyping)
